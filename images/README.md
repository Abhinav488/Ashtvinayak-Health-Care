# Image Placeholders for Ashtvinayak Health Care Website

This folder contains placeholder images. Replace them with real clinic photos for best results.

## Required Images

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `hero-bg.jpg` | Hero section background | 1920x1080 |
| `clinic-front.jpg` | Clinic exterior/front view | 1200x800 |
| `reception-area.jpg` | Reception/waiting area | 1200x800 |
| `doctor-room.jpg` | Doctor consultation room | 1200x800 |
| `equipment.jpg` | Medical equipment | 1200x800 |

## Image Guidelines

- Use high-quality images (minimum 1200px width)
- Ensure consistent lighting across images
- For the hero background, use a brighter image as it has an overlay
- All images should be in JPEG (.jpg) format
- File names must match exactly as shown above

## Notes

- The HTML includes fallback SVG placeholders that display when images are missing
- Images are lazy-loaded for better performance
- Alt text is included for SEO optimization
